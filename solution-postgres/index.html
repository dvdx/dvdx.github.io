  • Solution: Enhanced Postgres

Preamble

Postgres is a popular engine behind many database-driven applications today. Although most requests from the applications are short and sweet, longer duration and heavier queries are a fact of life that cannot be avoided. These hefty queries tend to obstruct and disrupt normal operation disproportionately because they perform big scan operations that:

  • expel hot pages from the cache,
  • monopolize system resources such as WAL, and
  • exacerbate lock contention.


Solution

Vitesse DB Standard Edition.

Vitesse DB Standard Edition is 100% compatible with Postgres, and heavy queries run on average 2X to 3X faster. Vitesse DB will free up resources sooner and shorten the window of disruption. Swapping out Postgres for Vitesse DB can be accomplished trivially without unload-reload of data. Customers will see that their machines load graphs got compressed after the swap (in tech-parlance, we say the database machine is happier after the swap):


Key Benefits

  • Reports are generated in a timely manner.
  • Web GUI is more responsive because all queries run faster when the machine is less loaded.
  • Hardware replacement cycle can be pushed out further.
